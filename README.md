# Game using Chainlink VRF at ETHGlobal Bangkok 2024

- ETHGlobal Bangkok from Nov 15, 2024 to Nov 17, 2024 
- Use Chainlink VRF v2.5 APIs


## Installation

```
$ cd backend
$ npm install
$ cd ../frontend
$ npm install
$ npm run build
```

## Run frontend
```
$ cd frontend
$ npm start
```
